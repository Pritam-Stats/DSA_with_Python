## Local Setup

1. Clone the repository:
```bash
git clone https://github.com/pritam-stats/DSA_with_Python.git
```

2. Navigate to the directory:
```bash
cd DSA_with_Python
```

3. Run any solution:
```bash
python3 path/to/solution.py
```
